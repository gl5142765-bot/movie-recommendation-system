# 🎬 Movie Recommendation System

## 🚀 Overview
This project is a content-based movie recommendation system built using linear algebra concepts and cosine similarity.

It recommends similar movies based on genre features.


## 🧠 Concepts Used
- Vectors & Feature Representation
- Cosine Similarity
- Matrix Operations
- Content-Based Filtering

---

## ⚙️ How It Works
1. Movie genres are converted into feature vectors.
2. Each movie is represented in vector space.
3. Cosine similarity is used to compare movies.
4. The system recommends top 5 similar movies.

---

## 💻 Example

Input:
recommend('Avatar')




Output:
Superman Returns
Man of Steel
X-Men: Days of Future Past
The Wolverine
